## 🔍 Key Features / 核心功能

- **🔍 Real-time Detection / 实时检测**
  Automatically identifies privacy information / 自动识别隐私信息

- **📊 Three-Level Warning System / 三级预警**
  - 🔴 Critical / 高危: API Keys, Passwords, Private Keys
  - 🟠 High / 中危: ID Cards, Bank Cards, Passports
  - 🟡 Medium / 低危: Phone Numbers, Emails, IP Addresses

- **🔒 Smart Obfuscation / 智能混淆**
  - Full masking: `****` / 完全遮盖
  - Partial masking: `138****8000` / 部分隐藏
  - Placeholder: `[Protected]` / 占位符
  - Fake data generation / 假数据生成

- **🎯 9 AI Platforms Supported / 支持9大AI平台**
  ChatGPT, Claude, Gemini, DeepSeek, Tongyi Qianwen, Wenxin Yiyan, Kimi, ChatGLM, iFlytek Spark

## 📋 Installation / 安装

### Method 1: Quick Install / 方式一：快速安装

1. Download `privacy-guard-v1.1.0.zip` / 下载压缩包
2. Extract to any folder / 解压到任意文件夹
3. Open Chrome/Edge extensions page / 打开扩展页面
4. Enable Developer Mode / 开启开发者模式
5. Load unpacked extension / 加载解压的扩展

**Detailed guide / 详细指南:** [INSTALL.md](INSTALL.md)

### Method 2: Source Install / 方式二：源码安装

```bash
git clone git@github.com:IntentionalBugs/privacy-guard.git
cd privacy-guard
# Load the folder in browser / 在浏览器中加载该文件夹
```
