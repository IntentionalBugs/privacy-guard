图标说明
========

当前使用的是占位图标。

请通过以下方式创建真实图标：

方法1：使用 generate-icons.html
  1. 在浏览器中打开 generate-icons.html
  2. 点击"下载全部图标"按钮
  3. 将下载的图标文件保存到此文件夹

方法2：使用任意图标设计工具
  创建以下尺寸的PNG图标：
  - icon16.png  (16x16像素)
  - icon48.png  (48x48像素)
  - icon128.png (128x128像素)

方法3：在线工具
  - https://www.favicon-generator.org/
  - https://realfavicongenerator.net/

推荐图标主题：
  - 盾牌形状
  - 紫色渐变背景 (#667eea 到 #764ba2)
  - 白色前景
